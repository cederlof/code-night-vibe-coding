# Snake game

Körde "Initial prompt.md" i M365 Copilot.

Gav följande spelidé: "A Snake game in the browser. The player is the snake. It can be single player or multiplayer (locally), but leave open possiblility of online multiplayer.""

Kopierade sen över md-text till respektive fil i ./spec/ och sen kickade igång kodgenereringen i Github Copilot (Claude Sonnet 4.5) med prompten: "Start implementing M1 only. Nothing else."
