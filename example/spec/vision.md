# Vision

A fast, grid-based Snake game playable in the browser.

## Player experience
- Immediate start: short countdown, then the snake auto-moves.
- Simple goal: eat food to grow and increase score; avoid death.
- Difficulty ramps via time-based speed increases.

## Modes (V1)
- Single-player first.
- Local multiplayer (2 players) as the next step.

## Success criteria for V1
- A complete, replayable round loop in ~90 seconds:
  - start → countdown → play → game over → restart
- Clear, readable HUD: score + speed level.
