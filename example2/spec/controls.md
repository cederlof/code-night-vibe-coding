# Controls

| Key | Action |
|-----|--------|
| **UP** | Accelerate forward |
| **DOWN** | Brake |
| **LEFT** | Turn left |
| **RIGHT** | Turn right |
| **R** | Restart maze |

## Notes
- Steering has drift/momentum (not instant turns)
- Holding UP continuously accelerates up to max speed
- Releasing UP causes gradual deceleration (friction)
